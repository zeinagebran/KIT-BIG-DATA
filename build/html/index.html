extract_and_load_data()

Extracts and loads recipes and interactions datasets from a zip file.

This method reads CSV files for interactions and recipes directly from the specified path and caches the results to optimize performance with Streamlit.

Returns:

A tuple containing two DataFrames:

  • interactions_df (pd.DataFrame): DataFrame containing interactions data.

  • recipes_df (pd.DataFrame): DataFrame containing recipes data.

Return type:

tuple

class functions.SeasonalityAnalysis(interactions_df)

Bases: object

A class to analyze and visualize seasonal trends in user interactions.

interactions_df

DataFrame containing interactions data with a ‘date’ column.

Type:

pd.DataFrame

plot_seasonality()

Plots the average user interactions by month and season.

This method processes the interactions DataFrame to extract information about the year, month, and season from the ‘date’ column, groups the data by month and season, and visualizes the average number of interactions for each.

Returns:

A matplotlib figure object displaying two bar plots, one for average

interactions per month and another for average interactions per season.

Return type:

matplotlib.figure.Figure
